# Meridian
Meridian is a global social platform built for the startup ecosystem — founders, investors, operators, and builders who think beyond borders. It's where serious people come to connect, share, and scale ideas that matter.Unlike generic professional networks, Meridian is built around the language of startups: traction, funding rounds, pivots, co-founders, and global markets. Every feature is designed to reduce noise and amplify signal, helping the right people find each other faster globally in an ever connected world.

This is Where the World Builds.
